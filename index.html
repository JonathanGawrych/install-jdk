  2. Open web browser "Internet Explorer"
  3. Go to the url http://www.oracle.com/technetwork/java/javase/downloads/java-archive-javase8-2177648.html#jdk-8u25-oth-JPR
  4. Click "accept license agreement" under Java SE Development Kit 8u25 section
  5. Click on jdk-8u25-windows-i586.exe for Windows x86
  6. A download box will appear saying if you want to run or save. Click run.
    If you see this image, you do not have the required administrator privileges
  7. The uac warning will appear. Click "yes" to allow the program.
  8. A setup window will appear. click the "next" button twice in the setup window. Click "next" for the destination as well
  9. Once the installer is finished, close the installer
  10. Open Notepad by hitting the "windows key" on your keyboard with "r" type "notepad" click "ok".
  11. Copy the following code: class HelloWorld{ public static void main(String args[]) { System.out.println("Hello World!"); } }
  12. Save the file to the desktop as HelloWorld.java.
    When saving make sure the option for "save as type" is set to "all files"
  13. Open command prompt by holding shift and right click on the Desktop. Click open command window here.
  14. Run the command "C:\Program Files (x86)\Java\jdk1.8.0_25\bin\javac" HelloWorld.java to compile the code
  15. Run the command "C:\Program Files (x86)\Java\jdk1.8.0_25\bin\java" HelloWorld to run the code

The end result should be a single line in the terminal that says Hello World!
